# 文档契约

每次工作流运行都必须在 `<project-root>/workflow/runs/<date>-<slug>/` 下使用这些契约。

规则：所有文档默认使用简体中文；代码标识符、命令、路径、错误输出、API 名称和用户原文可以保留原文。
